PMUT - INFO & COMMANDS

This text is meant for build 58 of PMUT

current commands :

print -a[ x... ]- prints out a string of characters(just words)
x... = how many words you want to put in there(to go in the next line just put comma)

exit - exits PMUT safely; recommended for exiting PMUT, becuase it closes & cleans up the librarys PMUT uses

print_shape -a[ x, y, z ]
x - Width
y - Height
z - character

get_version - gets the current version of PMUT

start_timer - starts stop watch

end_timer - ends stop watch

NEW COMMANDS :
NOTE : THESE SERVERS ARE ONLY ON THE LAN, NOT ON WAN(LOCAL ONLY)
//             VVV networking protocol, t
 make_server -t tcp -a[ x, y, z, w(optional) ]
x - IP of your server; note currently IPV4 adresses are the only acceptle ones
y - port number of your server
z - Name of your server
w(optional) - The first initial message sent to a client when one connects

run_server -t tcp -a[ x, y ]
x - The mode you want to run your server in, so single or multi, Single is the only one avalible
y - The name of the server you want to run
